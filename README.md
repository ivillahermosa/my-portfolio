# Personal Website Portfolio

Welcome to my personal website portfolio. This site showcase my skills, projects and experience as a Sotware Engineer / Fullstack .Net Developer

## Live Demo
Checkout the live site here: [https://ivanvillahermosa.com]



## Tech Stack
- **Frontend:** Next.js, React, Typescript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel
- **Other Tools:** Shadcn/ui, Resend, Upstash, ESLint, Prettier

## Features
- Full **responsive design** for desktop and mobile
- **Contact Form** integrated with email service
- **Dark/Light** mode toggle for user preference
- Smooth animations and transitions for improve UX

## Project Structure
- /app - App Router
- /components - Reusable React components
- /data - details including project list, skills, experiences
- /lib - utility functions and third party libraries
- /public - static assets (images, pdf)
- /types - interfaces (typescript)
